You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html


API’s To Try:

Combine Strings : 

Enter two strings to combine

http://52.15.227.232:9000/CombineStrings?Str1=Hello&Str2=World

Age Calculator :

Enter your date of birth and a reference date

http://52.15.227.232:9000/Age?DOB=2000-12-18&RefDate=2024-03-25
